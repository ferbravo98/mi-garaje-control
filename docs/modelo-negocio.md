# Modelo de Negocio

## Problema

Los usuarios suelen registrar mantenimientos vehiculares en papel, notas o memoria.

## Solución

Mi Garaje Control permite centralizar:
- vehículos
- mantenimientos
- gastos
- próximos services

en una única aplicación web responsive.

## Público objetivo

- Usuarios particulares
- Motociclistas
- Familias con múltiples vehículos

## Monetización futura

- Freemium
- Publicidad
- Suscripción premium
