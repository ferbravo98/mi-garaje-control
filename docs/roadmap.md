# Roadmap

## V1 - Versión inicial

Objetivo: construir una app funcional para uso personal desde navegador/celular.

### Funcionalidades

- Registro de vehículos
- Listado de vehículos
- Registro de mantenimientos
- Historial por vehículo
- Próximos mantenimientos
- Soporte español/inglés
- Base de datos SQLite
- Testing básico

## V2 - Mejoras funcionales

- Edición de vehículos
- Eliminación de vehículos
- Filtros por fecha
- Estadísticas de gastos
- Exportación CSV/PDF

## V3 - Versión avanzada

- Login de usuarios
- Backup en la nube
- Notificaciones
- Versión Android
- Compartir vehículo con otro usuario

## Alcance fuera de V1

Para evitar aumentar la complejidad inicial, la primera versión no incluirá:
- autenticación
- pagos
- sincronización cloud
- notificaciones push
- APK Android
