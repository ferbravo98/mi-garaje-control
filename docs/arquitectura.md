# Arquitectura del Proyecto

## Descripción general

Mi Garaje Control será una aplicación web responsive desarrollada con Python y Streamlit.

## Tecnologías

| Tecnología | Uso |
|---|---|
| Python | Lenguaje principal |
| Streamlit | Interfaz web |
| SQLite | Base de datos local |
| Pandas | Manejo de datos |
| Pytest | Testing |
