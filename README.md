# Mi Garaje Control 🚗🏍️

Aplicación web responsive para gestión de mantenimiento vehicular.

## Características

- Registro de vehículos
- Historial de mantenimientos
- Control de kilometraje
- Próximos services
- Soporte español / inglés
- Responsive para celulares

## Tecnologías

- Python
- Streamlit
- SQLite
- Pandas
- Pytest

## Roadmap

### V1
- Gestión de vehículos
- Historial de mantenimientos
- Próximos services

### V2
- Login de usuarios
- Backup cloud
- APK Android
